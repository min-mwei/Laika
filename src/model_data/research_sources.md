# Seed Question Research

Sources consulted to expand web navigation and form interaction question templates:

1) Playwright Actions docs (text input, checkboxes/radios, select options, clicks)
- https://playwright.dev/docs/input

2) MDN "Your first form" (form controls, labels, input/email/textarea/button)
- https://developer.mozilla.org/en-US/docs/Learn/Forms/Your_first_form

3) Selenium "Interacting with web elements" (click, send keys, clear)
- https://www.selenium.dev/documentation/webdriver/elements/interactions/

Derived seed question themes:
- Fill text inputs (name/email/message) and submit forms.
- Select dropdown options and toggle checkboxes.
- Click primary buttons or first links.
- Search within a page when a search field exists.
- Summarize or describe the first/second topic.
